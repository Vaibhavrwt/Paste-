# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.


# PASTE 📝

PASTE is a modern and responsive paste management web application built using **React.js** and **JavaScript**.  
It allows users to create, save, edit, update, copy, share, view, and delete pastes effortlessly.

The application stores all paste data locally using **Local Storage**, ensuring quick access and persistence without requiring a backend.

---

## 🚀 Features

- ✍️ Create new pastes
- 📝 Edit & update existing pastes
- 👀 View saved pastes
- 📋 Copy paste content instantly
- 🔗 Share pastes easily
- 🗑️ Delete unwanted pastes
- 📅 Automatically saves creation date & time
- 💾 Local Storage support for persistent data
- 🎨 Fully styled using Tailwind CSS
- 📱 Responsive & mobile-friendly UI
- ⚡ Fast and lightweight React application

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- Tailwind CSS
- Local Storage API

💡 How It Works
Users can create and save text-based pastes.
All pastes are stored inside the browser using Local Storage.
Existing pastes can be edited or updated anytime.
Users can copy or share paste content easily.
Each paste includes a saved creation date.
